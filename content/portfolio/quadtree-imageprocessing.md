## About

A command-line image processing tool built in Java using a quadtree data structure. Supports compression at configurable thresholds, edge detection, quadtree outline visualization, color inversion, and shading. My first substantial solo programming project, built freshman year.

## Timeline and scope

Fall 2022. Solo project for an introductory CS course at Bryn Mawr College.

## Skills and technologies

Java, quadtree data structures, PPM image format parsing, recursive algorithms
