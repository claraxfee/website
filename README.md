
# my personal website built with Hugo and the PaperMod theme, deployed via GitHub Pages

live site: https://clarafee.xyz

## setup

- Hugo Extended v0.125.7+ (required for PaperMod)
- Git with SSH keys set up for GitHub

```bash
# latest Hugo Extended from GitHub releases
wget https://github.com/gohugoio/hugo/releases/download/v0.139.3/hugo_extended_0.139.3_linux-amd64.tar.gz

tar -xvf hugo_extended_0.139.3_linux-amd64.tar.gz
mkdir -p ~/.local/bin
mv hugo ~/.local/bin/

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

hugo version
```

```bash
git clone git@github.com:claraxfee/website.git
cd website

# Initialize PaperMod submodule
git submodule update --init --recursive
```

## deployment

deployment fully automated via GitHub Actions. on push to `main`:

1. GitHub Actions runs the workflow in `.github/workflows/hugo.yml`
2. Hugo builds the site with `hugo --minify`
3. Site deploys to GitHub Pages
4. Available at https://clarafee.xyz within ~2 minutes

Check deployment status: https://github.com/claraxfee/website/actions

## file structure

```
website/
├── config.yml              # site config (title, menu, params)
├── content/                # my content, edit here
│   ├── about.md
│   ├── blog/              # blog posts
│   └── portfolio/         # portfolio items
├── static/                # static files (images, CNAME)
│   └── images/
├── assets/
│   └── css/
│       └── extended/
│           └── custom.css  # my custom CSS
├── themes/
│   └── PaperMod/          # DON'T EDIT (git submodule)
└── public/                # DON'T EDIT (generated by hugo)
```

### can edit:

- **`content/`** - blog posts, portfolio items, about page
- **`static/images/`** - images
- **`assets/css/extended/custom.css`** - custom CSS
- **`config.yml`** - site settings, menu, social links, etc.


## customizing theme w hugo's overriding system

1. Copy from `themes/PaperMod/layouts/` to root `layouts/` folder (same path)
2. Edit the copy - it takes precedence over theme version

example - customizing the footer:

```bash
mkdir -p layouts/partials
cp themes/PaperMod/layouts/partials/footer.html layouts/partials/footer.html
# now edit layouts/partials/footer.html
```

**For custom CSS:** just edit `assets/css/extended/custom.css` (no override needed)



## troubleshooting log

### localhost shows old content

1. Stop hugo server (Ctrl+C)
2. Delete `public/` folder: `rm -rf public/`
3. Restart: `hugo server`
